### MovieBox ###

# Project Overview
This is a responsive movie website built using HTML, CSS, and JavaScript.
The project focuses on modern UI design, smooth user interaction, and full compatibility with different screen sizes.

# Project Description
The website provides a clean and user-friendly interface for browsing movies and series.
It includes multiple sections, interactive elements, and responsive navigation designed to work seamlessly on desktop, tablet, and mobile devices.

# Key Features
1. Fully responsive design for all screen sizes
2. Modern and clean user interface
3. Interactive movie cards with hover effects
4. Mobile-friendly navigation menu
5. Toggleable search bar
6. Organized layout using CSS Variables and Media Queries

#Technologies Used
1. HTML5
2. CSS3
3. JavaScript (Vanilla JavaScript)
4. Ionicons
